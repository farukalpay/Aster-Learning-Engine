# Mesh Lab

Mesh Lab proves that the same render contract handles authored primitives and
procedural/custom meshes.

Scene:

- `mesh-lab`: terrain patch, twisted cable, fractured proxy mesh, and cave web.

Preview:

```bash
./build/aster_preview --scene mesh-lab --output /tmp/aster_mesh_lab.ppm --width 1280 --height 720 --samples 2
```
